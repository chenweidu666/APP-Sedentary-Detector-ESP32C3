# PM11 锂电池包

[← 硬件文档索引](../HARDWARE.md)

```
尺寸: ~40 × 20 × 12mm (含外壳)
接口: JST-PH 2.0 (输出 5V/2.4A)

功能:
  - 3.7V LiPo 电池内置
  - TP4056 充电电路 (Type-C 充电)
  - 升压模块 5V/2.4A 输出
  - 低电量保护
```

## 本设计使用

- 5V 输出 → 滑动开关 → ESP32 VIN (Pin 15)
- GND → ESP32 GND
- 充电口外露，无需载板走线

详细供电拓扑见 [`system/power.md`](../system/power.md)。
