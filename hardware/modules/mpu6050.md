# GY-521 模块 (MPU6050)

[← 硬件文档索引](../HARDWARE.md)

```
尺寸: 15.6 × 20.2 mm
重量: ~15g
引脚: 1 × 8pin 单排, 2.54mm 间距

      ┌──────────────┐
      │              │
      │  [MPU6050]   │  ← 15.6mm
      │   GY-521     │
      │              │
      └─┬─┬─┬─┬─┬─┬─┬┘
        │ │ │ │ │ │ │ │
       VCC GND SCL SDA XDA XCL AD0 INT
       ↑
       20.2mm
```

## 本设计使用 (5 根)

| 模块 Pin | 信号 | 连接 |
|----------|------|------|
| 1 | VCC | ESP32 3V3 (Pin 1) |
| 2 | GND | ESP32 GND |
| 3 | SCL | ESP32 GPIO5 (Pin 22) |
| 4 | SDA | ESP32 GPIO4 (Pin 25) |
| 8 | INT | ESP32 GPIO2 (Pin 26) |

未使用: XDA, XCL, AD0（AD0 默认接 GND，I2C 地址 0x68）
