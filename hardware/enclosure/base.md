# 3D 底座配合

[← 硬件文档索引](../HARDWARE.md)

载板 PCB 四角 M3 安装孔与 3D 打印底座对应:

```
底座截面 (侧视):

    ┌─ 载板 PCB ──────────┐
    │  (M3 螺丝固定)       │
 ───┴──────────────────────┴───  ← 底座上表面
 │         LiPo 电池            │  ← 电池仓 (底座内)
 │         (扁平软包)            │
 └──────────────────────────────┘  ← 底座底面 (贴椅子)
         ↑ 双面胶/扎带固定到椅子
```

## 底座内部预留

- 电池仓: 50 × 30 × 8mm (适配 1000mAh 扁平 LiPo)
- PCB 固定柱: 4 个 M3 铜柱, 高度 10mm (电池上方)
- Type-C 开口: 侧面留缺口
- 开关开口: 侧面留缺口
